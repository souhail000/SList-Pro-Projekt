# SList-Pro

SList-Pro ist eine smarte Einkaufslisten App, zur Erstellung kompakter Listen. Nutzer können Artikel inklusive Menge und Priorität in Listen erfassen,sowie bearbeiten, oder löschen. Einträge können anschließend als "gekauft" markiert werden

# Features
- Benutzer können Einkaufslisten erstellen
- Artikel hinzufügen und bearbeiten
- Artikel als gekauft markieren
- einzelne Artikel/ ganze Listen löschen
- Prioritäten der Artikel setzen
